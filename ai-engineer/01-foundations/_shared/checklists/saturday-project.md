# Checklist Sábado - Projeto da Semana

> Modelo: brief → implementação → ditado → checklist

## 1. BRIEF (5 min)
- [ ] Ler objetivo do projeto da semana
- [ ] Revisar critérios de pronto
- [ ] Copiar estrutura de pastas padrão

## 2. IMPLEMENTAÇÃO (90 min)
- [ ] Criar arquivos do projeto
- [ ] Implementar funcionalidade principal
- [ ] Escrever testes básicos
- [ ] Fazer funcionar (mínimo viável)

## 3. DITADO (10 min)
- [ ] Gravar README usando áudio → texto
- [ ] Incluir: o que faz / como rodar / resultado
- [ ] Revisar rapidamente (não precisa ser perfeito)

## 4. CHECKLIST FINAL (5 min)
- [ ] Rodar testes
- [ ] Gerar outputs (se aplicável)
- [ ] Commit no GitHub
- [ ] Marcar critérios de pronto

---

## Tempo total: ~2h
