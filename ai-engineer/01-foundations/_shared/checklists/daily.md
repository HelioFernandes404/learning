# Checklist Diária (Seg-Qui)

> Bloco de 2 horas - formato sempre igual

## 1. INPUT (30 min)
- [ ] Assistir vídeo curto OU ler exemplo mínimo
- [ ] Anotar 3 perguntas-chave no `notes.md`
- [ ] TTS (texto para fala) se preferir ouvir

## 2. BUILD (70-80 min)
- [ ] Implementar SEM copiar/colar
- [ ] Rodar pelo menos 1 teste (mesmo simples)
- [ ] Fazer funcionar antes de otimizar

## 3. RETRIEVAL (10 min)
- [ ] Fechar TUDO (editor, navegador)
- [ ] Escrever no `retrieval.md`: "Como eu faria isso do zero?"
- [ ] Marcar ⚠️ nas lacunas

## 4. LOG (5 min)
- [ ] Atualizar `learning-log.md` com 2 linhas (pode ditar)
- [ ] Formato: Data | Tema | O que fiz | Próximo passo

---

## Tempo total: ~2h
